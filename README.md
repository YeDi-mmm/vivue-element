# Vue 3 + Vite 4

+ 安装依赖：npm install
+ 快速启动：npm run dev
+ 项目打包：npm run build

## 技术栈

+ ⚡️ Vite 4 - 构建工具（就是快！）
+ 🖖 Vue 3 - 渐进式 JavaScript 框架
+ 🚦 Vue Router 4 - 官方路由管理器
+ 📦 Pinia - 值得你喜欢的 Vue Store
+ 🔗 Axios - 一个基于 promise 的网络请求库，可以用于浏览器和 node.js
+ 🛡️ ESLint - 代码规范
+ 🌏 vue-i18n - 语言国际化
+ 💻 element-plus - UI库，能够快速开发
